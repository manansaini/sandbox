HTTP Status Code is returned when an HTTP request is made to the server. The server returns an HTP Status Code in the response to your request. There are five classes of HTTP Status Code as following.

    Informational – 1XX
    Success – 2XX
    Redirection – 3XX
    Client Error – 4XX
    Server Error – 5XX
